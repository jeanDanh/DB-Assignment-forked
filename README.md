# DB assignment


| File name                  | Description                                                                                                         |
|----------------------------|---------------------------------------------------------------------------------------------------------------------|
| `schema.sql`               | It setups tables, basic triggers.                                                                                   |
| `insert.sql`               | It populates tables with dummy data.                                                                                |
| `drop_all.sql`             | It provides handy dropping commands.  You use in case you want to delete  everything and start over.                |
| `functions.sql`            | It provides all functions. Functions perform sophisticated series of actions and eventually return a value.         |
| `procedure.sql`            | It provides all procedures. Procedures perform sophisticated series of actions but not supposed to return anything. |
| `semantic_constraints.sql` | It provides all triggers which used to check semantic constraints.                                                  |
|                            |                                                                                                                     |
